# Realtime AI Fraud Detection System

This is a real-time AI-powered fraud detection engine with AML explainability.

## How to Run

pip install -r requirements.txt  
python fraud_engine.py  
streamlit run app.py

## Features
• Real-time transaction streaming  
• Fraud scoring system  
• AML rule-based explainability  
• Live dashboard
